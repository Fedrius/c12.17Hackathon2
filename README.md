# Surfs Up

Application to show you weather and surf conditions at any beach!

## Technologies Used

### Front end:

>HTML, CSS, Javacript, Bootstrap, and jQuery.

### Back end:

>Node.js, Express, Passport.js, and MySql.

### APIs:

>Google Geo Location, Flikr, and World Weather Online.
